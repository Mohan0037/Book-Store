# BookStore - MERN Stack Online Book Shopping System

## Project Overview

BookStore is a full-stack online book shopping application developed using the MERN Stack (MongoDB, Express.js, React.js, Node.js).

The platform provides separate portals for Readers, Sellers, and Administrators. Readers can browse books, search titles, add books to their cart, and place orders. Sellers can add and manage books with cover images, while administrators manage platform operations.

The application provides a responsive and user-friendly experience across different devices.

---

# Features

## Reader Portal

- User registration and login
- Browse available books
- View book details
- Search and explore books
- Add books to cart
- Place orders
- View order history

---

## Seller Portal

- Seller registration and authentication
- Seller dashboard
- Add new books
- Upload book cover images
- Manage uploaded books
- Track seller products

---

## Admin Portal

- Admin authentication
- Admin dashboard
- Manage application data
- Monitor users and sellers
